# 자료 의미와 계산 기준

## 공개 지표

| 화면 지표 | 내부 지표 | 의미 |
| --- | --- | --- |
| 최고기온 | `tasmax` | 일 최고기온 |
| 최저기온 | `tasmin` | 일 최저기온 |
| 강수량 | `pr` | 일 강수량 |
| 풍속 | `wind_speed` | 일 풍속. `sfcWind`를 우선하고 필요한 경우 `uas`, `vas`로 계산 |
| 열지수 | `heat_index` | 고온·다습 조건에서의 열 스트레스 지수 |
| 체감기온 | `feels_like` | 더위, 추위, 바람을 계절에 맞게 반영한 일 대표 체감기온 |

## 열지수와 체감기온의 차이

`열지수`와 `체감기온`은 같은 값이 아닙니다.

1. 열지수는 기온이 80°F 이상이고 상대습도가 40% 이상인 날에 NWS Rothfusz 회귀식을 적용합니다.
2. 체감기온은 열지수가 유효하면 열지수를 사용합니다.
3. 열지수가 없고 최저기온이 10°C 이하이며 풍속이 4.8km/h를 넘으면 풍속냉각지수를 사용합니다.
4. 두 지수가 모두 적용되지 않으면 일 최고·최저기온의 평균을 사용합니다. 한쪽만 있으면 존재하는 기온을 사용합니다.
5. 필요한 원자료가 모두 없으면 값을 만들거나 직선 보간하지 않고 결측으로 유지합니다.

따라서 열지수 그래프에는 계절에 따른 결측 구간이 자연스럽게 존재할 수 있지만, 체감기온은 기온 원자료가 있는 날에는 계속 제공됩니다. 그래프는 결측 구간을 직선으로 연결하지 않습니다.

## 앙상블

전체 앙상블은 날짜별로 값이 있는 모델만 사용해 p10, p50, p90을 계산합니다. 사용 가능한 모델 수는 날짜와 지표마다 달라질 수 있으며, UI와 CSV의 `model_count`에 기록됩니다.

## 원자료와 보정값

- 기준 지점 범위 안에서는 제공되는 보정값과 원자료를 구분합니다.
- 범위 밖 좌표는 CMIP6 원자료 격자를 조회하며 관측자료 기반 보정을 적용한 것처럼 표시하지 않습니다.
- CSV의 결측값은 빈 칸과 `coverage=missing`으로 기록합니다.
